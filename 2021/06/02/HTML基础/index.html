标签的语法

  • 标签由英文<>括起来，例如：

    <html>

  • 标签一般成双出现，例如：

    <p></p>  <h1></h1> <div></div> <span> </span>

  • 标签不区分大小写，但大部分程序员用小写。

HTMl文档结构

 <html></html>标签对：<html>位于文档的最前面，用来标识文档的开始；</html>用于文档的最后面，用来标识文档的结束。中间部分是文档的头部（head）和主体（body）。

<head></head>标签对：包含一些文档的信息，可以包含文档的辅助性标签，
例如：<title></title>:标题

     <style></style>:用来修改文件的样式

     <script></script> 

     <link/> <meta/>
  
     <meta charset="UTF-8">:设置当前文件字符编码

 但是浏览器只会显示<title>的内容。

<body></body>:中间放html的主要内容。

注释文字：<!--注释文字-->

各种标签

语义化标签

  • 段落标签：

    <p> 段落文本</p>

  • 自定义样式标签:文字

    需要在head标签中使用style标签：

    <head>
    <meta charset="UTF-8">
    <title>了不起的盖茨比</title>
    <style>
    span {
        color:blue;//将字体设置为蓝色
    }
    </style>
</head>

  • 标题标签：

    <h1></h1> <h2></h2>....共有6级，字号逐级递减

  • 为网页划分独立板块：

    <div>内容</div> 相当于一个容器，将内容放在里面，将各个部分划分开。

  • 头部标签：

    <header>我是头部标签</header>

  • 底部标签：

    <footer> 我是底部标签</footer>

  • 正文标签：

    <section>我是一个区域</section>

  • 侧边栏标签：

    <aside>我是侧边栏</aside>

    效果标签

  • 换行标签：

    <br />直接在换行的部分输入

  • 空格标签：

    &nbsp

  • 水平分割线：

    <hr />

    列表标签

  • 无序列表：

    #<ul><li>内容<li><ul>
    
<body>
    <ul>
        <li> 我的第一个列表信息</li>
        <li>我的第二个列表信息</li>
    </ul>
</body>

  • 有序列表：

    #<ol><li>内容<li><ol>

<body>
<ol>
    <li>我的第一个列表信息</li>
    <li>我的第二个列表信息</li>
</ol>
</body>

  • 为网页添加图片

    <img src="E:/HTML/笔记/图片的地址" alt="图片下载失败时的替换文本" title="提示文本" />

src:图像的位置   alt:替换文本  title:当鼠标划过该图片时显示的文字  图像可以是GIF、PNG、IPEG格式

  • 网页链接

    <a href="网址"  title="提示文本">链接显示的文本</a>

  • 在浏览器窗口打开链接

    <a href="http://www.baidu.com" target="_self">百度</a>:target="_self"在当前页面打开链接

target=”_blank":新建窗口打开链接

  • 表格标签

    <table></table>:用来定义整个表格

border="1":用来加边框

<caption></caption>:表格的标题

<tr></tr>表示一行

<th></th>:一列，有加粗效果

<td></td>:一列，无加粗效果

    例如：

    <html>

<head>
    <meta charset="UTF-8">
    <title>学习表格标签</title>
</head>

<body>
    <table border="1">
        <caption>前端三剑客</caption>
        <tr>
            <th>知识点</th>
            <th>重要程度</th>
            <th>难度</th>
            <th>学习周期</th>
        </tr>
        <tr>
            <td>html</td>
            <td>5星</td>
            <td>3星</td>
            <td>7天</td>
        </tr>
        <tr>
            <td>css</td>
            <td>5星</td>
            <td>4星</td>
            <td>10天</td>
        </tr>
        <tr>
            <td>js</td>
            <td>5星</td>
            <td>5星</td>
            <td>20天</td>
        </tr>
    </table>

    使用thead、tbody、tfooyt定义表格

    1.<thead>标签表示表格头部,一般放<tr>和<th>标签。

2.<tbody>标签表示表格内容,一般放<tr>和<td>标签。

3、<tfoot>标签标签表格脚部标注,一般放<tr>和<td>标签。

4、对表格中的行进行分组，这种划分使浏览器有能力支持独立于表格标题和页脚的正文滚动，当长的表格被打印时，表格的表头和页脚也就可被打印在包含表格数据的每张页面上。

    表单标签

    1.表单是可以把浏览器输入的数据传送到服务器端，这样服务器端的程序就可以处理表单传过来的数据。

    2.语法:

    <form method="传送方式"  action="服务器文件"> </form>

    3.用户名密码：

    type:

    • text：输入框为文本输入框
    • password：密码输入框

    name:为文本框命名

    value:为文本框设置默认值，一般起到提示作用

    <form>
    <input type="text/password" name="名称" value="文本" /> </form>

    1. 给点提示：placeholder属性的使用

      placeholder属性为输入框占位符，里面可以放提示的输入信息。

      html>

<head>
    <meta charset="UTF-8">
    <title>input-placeholder</title>
</head>

<body>
    <input type="text" placeholder="请输入用户名" >
    <br/>
    <input type="password" placeholder="请输入密码">
</body>

</html>

    2. 数字输入框

      type=”number”数字框只能输入数字，最后面有个加减号用来调整数字大小。

    3. 网址输入框

      type=”url” 内容必须以http：或者https：开头

    4. 邮箱输入框：type=”email”

    5. 创建文本域：

      <textarea cols="行数“ orws="列数” >在这里输入内容</textarea>
      <body>
    <form>
        <textarea cols="50" rows="10">在这里输入内容...</textarea>
    </form>
</body>

10.单选框和复选框

  • type：当type=”radio”时，控件为单选框

    ​ 当type=”checkbox”时，控件为复选框

  • value:提交到后台服务器，共后台使用

  • name：为控件命名，以备后台程序ASP、PHP使用，单选时选项的名字必须相同

  • 当设置checked=”checked”时，该选项被默认选中

​

11.创建下拉菜单

  • select和option标签都是双标签

  • select标签内只能放option标签，表示下拉菜单的选项

  • option标签放选型内容，不放置其他标签

  • value：提交到服务器的值

  • 当设置成selected=”selected”时，该选项被默认选中

    <body>
   <form> 
       <select>
         <option value="看书">看书</option>
         <option value="游戏" selected="selected">游戏</option>
           </select>
    </form>
</body>
